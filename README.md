SkillLaunch landing page is a modern, responsive landing page built as part of The Odin Project Foundational Course. SkillLaunch is a fictional online learning platform that helps beginners become web developers through hands-on projects and structured learning.

![SkillLauch homepage] (./screenshots/skilllaunch-homepage.png)

Features of SkillLaunch landing page includes: Responsive layout for desktop and mobile  viewpoint, Modern hero section, feature cards, Testimonial sections, Call-to-action section, Footer, Hover effects on buttons and clean flexbox layout.
SkillLaunch is built with HTML5, CSS3, Flexbox, Media Queries and Google fonts.
While building this projects i practiced structuring semantic HTML, Building responsive layout with Flexbox, Using media queries for mobile devices, Styling buttons and hover effect, Working with spacing, containers, and max-width, and creating reusable CSS classes.
In future i hope to add javaScript interactions, smooth scrolling navigation, animations, dark mode toggle and improve accesibility.
